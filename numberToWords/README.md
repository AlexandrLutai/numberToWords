# Преобразование чисел в слова

Этот проект содержит функции для преобразования чисел в их словесное представление на русском языке. Основные файлы - `numberToWord.cpp` и `numberToWord.h`.

## numberToWord.cpp

### Константы

- `valutes`: Массив строк, представляющих различные формы слова "Рубль" в зависимости от числа.
- `ranks`: Массив строк, представляющих различные разряды (тысячи, миллионы, миллиарды) в различных формах.
- `tensInWords`: Массив строк, представляющих десятки.
- `hundredsInWords`: Массив строк, представляющих сотни.
- `thousandExeption`: Массив строк, представляющих исключения для тысяч.
- `tensExeption`: Массив строк, представляющих исключения для десятков.
- `unitsInWords`: Массив строк, представляющих единицы.
- `digitsInWordsArrays`: Массив указателей на массивы строк для различных представлений цифр.

### Функции

#### `int parseRightDigit(unsigned long* number)`

Парсит крайнюю правую цифру из числа и возвращает её.

- **Параметры**:
  - `number`: Указатель на число для парсинга.

#### `std::string getUnitName(unsigned short unit, unsigned short rank, bool isTenExeption)`

Возвращает словесное представление единицы.

- **Параметры**:
  - `unit`: Цифра единиц.
  - `rank`: Разряд цифры.
  - `isTenExeption`: Булево значение, указывающее, является ли число исключением (10-19).

#### `std::string getTenName(unsigned short tens, unsigned short rank, bool isTenExeption)`

Возвращает словесное представление десятков.

- **Параметры**:
  - `tens`: Цифра десятков.
  - `rank`: Разряд цифры.
  - `isTenExeption`: Булево значение, указывающее, является ли число исключением (10-19).

#### `std::string getHundredName(unsigned short hundreds, unsigned short rank)`

Возвращает словесное представление сотен.

- **Параметры**:
  - `hundreds`: Цифра сотен.
  - `rank`: Разряд цифры.

#### `std::string getRankName(unsigned short unit, unsigned short rank, bool tenExeption)`

Возвращает словесное представление разряда.

- **Параметры**:
  - `unit`: Цифра единиц.
  - `rank`: Разряд цифры.
  - `tenExeption`: Булево значение, указывающее, является ли число исключением (10-19).

- **Объяснение формулы**:
  - `elementPosition = (countElementsInRank * rank) / 2 + unit * !tenExeption`
  - `countElementsInRank`: Количество элементов в массиве разрядов.
  - `unit`: Цифра единиц.
  - `tenExeption`: Булево значение, указывающее, является ли число исключением (10-19).
  - `rank`: Разряд цифры.
  - `2`: Константа, используемая для корректного вычисления позиции элемента.

#### `std::string getNumberAsWords(unsigned long number)`

Преобразует число в его словесное представление.

- **Параметры**:
  - `number`: Число для преобразования.
